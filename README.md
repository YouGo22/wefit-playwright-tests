# wemovies-e2e

Testes E2E com Playwright + TypeScript para o desafio WeFit (WeMovies) de Yugou Gombata.

## Cenários automatizados (3)
1. Adiciona e/ou remove um filme do carrinho.
2. Adicionar um filme ao carrinho e verificar subtotal/total.
3. Finalizar compra e verificar tela de confirmação.

## Pré-requisitos
- Node.js
- npm

## Instalação
```bash
git clone <SEU_REPO_URL>
cd wemovies-e2e
npm install
npm run install:browsers  # instala navegadores do Playwright
